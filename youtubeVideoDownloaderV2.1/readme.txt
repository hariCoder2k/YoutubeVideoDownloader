To use the application
Ensure pip and python are available in your pc. 
First run the command below and ensure its' successful
	pip install -r requirements.txt

To run the application
	python3 youtubeDownloaderV2.py

Enter the link of the youtube video to download and path to save the file.

Limitation

* Age Restricted videos cannot be downloaded.
